# Time Off

## Requirements
  * PostgreSQL

## Setup
  1.  Set environment variables
    * `TIME_OFF_CLIENT_ID`
    * `TIME_OFF_CLIENT_SECRET`
  2. `bundle install`
  3. `rake db:create`
  4. `rake db:migrate`

## Running the tests
  * `rspec`

## Running the app
  * `rails s`
